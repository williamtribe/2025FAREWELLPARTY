# 2025FAREWELLPARTY
2025 송년회의 웹사이트
.env에 필요한 값들은 모두 존재
REACT와 KAKAO login을 이용해서 카카오톡으로 로그인하고 정해진 양식에 따라 자신의 소개를 입력할 수 있는 페이지 각자 제공 (단, 관리자 계정은 모든 페이지에 접근 가능)
각 회원은 자신의 웹페이지를 보고 편집할 수 있고, 공개범위를 설정할 수 있음
정해진 양식에 따라 받아진 그들을 분석하는 텍스트는 결국 OpenAI의 3072차원 large-embedding-model에 의해 벡터화되고 Pinecone에 저장된다.
이에 따라 이들은 비슷한 가치관, 성격, 취향, 전문분야와 같은 다양한 기준으로 clustering 된다.
또한, Tinder의 library를 이용해서 이들에게 지속적으로 취향을 물어볼 수 있는 설문을 만들고, 그것 또한 데이터로 활용한다.
